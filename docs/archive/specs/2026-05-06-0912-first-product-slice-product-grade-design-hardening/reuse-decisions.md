## Decisions that supersede archived design-hardening

- Recording overlay 不再接受普通 Radix Dialog 作为最终产品形态；必须采用 Drawer/Vaul 或有证据的 fallback。
- Waveform 不再接受自研轻量 bars 作为最终产品形态；必须优先采用 ElevenLabs UI Waveform source，并按 Reo token 裁剪。
- Sidebar 不再从 first slice 移除；至少 covered/expanded/resizing layered shell 和 Home/New memory/Search 结构必须 high-fidelity。
- Films/photo/video 不能当前实现，但必须在 wireframe 中被设计，不得在产品实现里露出未实现能力。
