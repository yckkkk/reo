## 自研例外

当前允许的自有代码只限：

- Feature-local recording reducer 和 mock transcript ticker。
- Reo-specific workspace file templates。
- Reo-specific filesystem transaction orchestration。
- Reo-specific UI composition around adopted primitives。

这些都不是 generic runtime 或 reusable platform layer。
