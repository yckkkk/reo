# 安全威胁模型

本 threat model 按 asset 做 STRIDE。Codex CLI read-only validation 是 QA evidence，不是 security control。

| 资产                           | 边界                                 | 威胁                        | 攻击路径                                                | 缓解/预防                                                                                                                    | 检测/恢复                              | 负向测试/手动验证                                | 负责人        | 剩余风险                                             |
| ------------------------------ | ------------------------------------ | --------------------------- | ------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | -------------------------------------- | ------------------------------------------------ | ------------- | ---------------------------------------------------- |
| Renderer window                | untrusted DOM 到 privileged app      | Spoofing / Elevation        | XSS 尝试访问 Node/Electron                              | sandbox、contextIsolation、nodeIntegration false、不暴露 Electron APIs                                                       | runtime check `window.require` absent  | Electron runtime smoke                           | Electron      | XSS 仍可篡改 renderer 内 UI                          |
| Preload bridge                 | isolated preload 到 renderer         | Elevation                   | 暴露 generic `ipcRenderer.invoke` 或 fs                 | `contextBridge` 只暴露产品方法                                                                                               | preload bridge tests 枚举 exposed keys | leak tests                                       | Electron      | API surface 会随 feature 增长                        |
| IPC channel                    | renderer 到 main                     | Spoofing / Tampering        | untrusted frame invoke channel                          | 主 frame、trusted origin、session binding、channel allowlist、handle ownership validation                                    | rejected sender tests                  | fake sender、subframe、wrong session tests       | Electron      | Electron API shape drift                             |
| IPC payload                    | renderer 到 main                     | Tampering / DoS             | malformed payload、large JSON/binary                    | Zod parse、1 MiB chunk limit、1 in-flight append、binary sequence checks                                                     | structured error envelope              | schema fuzz/boundary/budget tests                | Data/Quality  | first slice limits may need later product adjustment |
| Workspace handle               | renderer capability 到 main registry | Elevation / Tampering       | renderer 伪造 rootPath 或复用其他 window handle         | main-owned `workspaceHandle`，绑定 canonical realpath、sender、workspaceId、lock owner                                       | `ERR_HANDLE_INVALID`                   | forged handle、cross-window handle tests         | Electron/Data | app restart 后 handle 不持久化，需 reopen            |
| Workspace folder               | user filesystem                      | Tampering                   | recordingId path traversal、symlink escape、TOCTOU swap | root/parent `realpath`、per-segment `lstat`、exclusive temp create、rename target symlink rejection                          | path traversal error                   | traversal、symlink parent、temp swap tests       | Main/Data     | network FS semantics may vary                        |
| Workspace folder               | Reo 到 user files                    | Tampering                   | 覆盖已有 `AGENTS.md` 或用户文件                         | 如果 `AGENTS.md` 已存在且无 workspace metadata，则 fail closed；exclusive create                                             | conflict 时不写文件                    | conflict fixture                                 | Main/Data     | 用户可能手动改文件                                   |
| Workspace lock                 | Reo processes                        | DoS / Tampering             | 两个 window 同写一个 workspace                          | `proper-lockfile` thin adapter、10s heartbeat、60s stale threshold、owner pid check                                          | locked error、recovery action          | duplicate open、stale lock、owner pid dead tests | Main/Data     | stale lock false positives                           |
| `.reo/workspace.json`          | disk 到 main                         | Tampering                   | unsupported version 或 invalid JSON                     | Zod parse、version gate、no silent migration                                                                                 | unsupported schema error               | corrupt metadata fixture                         | Data          | 用户编辑可能阻塞 workspace                           |
| `.reo/index.json`              | disk 到 UI                           | Tampering                   | corrupt index hides entries                             | index 可从 recordings 重建                                                                                                   | open 时 rebuild                        | corrupt index test                               | Data          | 后续数据量大时 rebuild 可能慢                        |
| Recording metadata             | disk 到 main                         | Tampering                   | recording 指向 workspace 外                             | relative refs only、containment check                                                                                        | entry marked unsupported               | malicious recording.json fixture                 | Data          | 可能需要手动恢复                                     |
| Audio artifact                 | browser/main                         | DoS                         | 超长 audio 或 chunk flood                               | chunk sequence、1 MiB chunk、1 in-flight append、60 min/120 MiB cap、append fsync、UI status                                 | append failure + draft recovery        | chunk flood、long recording smoke                | Frontend/Main | limits are first-slice safety budget                 |
| Audio artifact                 | renderer playback                    | Information disclosure      | Blob URL 泄漏或未 revoke                                | 只为 active playback 创建，close/switch/unmount 时 revoke                                                                    | URL revoke test                        | playback tests                                   | Frontend      | overlay 打开期间 Blob 必然存在                       |
| Media permission               | renderer 到 OS                       | Elevation                   | 从非 trusted URL 请求 video/camera 或 mic               | session permission handlers 只允许 trusted audio                                                                             | permission denied                      | permission tests + runtime                       | Electron      | Chromium permission UX 差异                          |
| Custom protocol                | URL 到 local file                    | Information disclosure      | `reo-app://renderer/../../secret`                       | scheme registered before ready、host `renderer` only、renderer build path containment、不用 app protocol 服务 user workspace | 400/404 response                       | protocol traversal、host、MIME、CSP tests        | Electron      | future media protocol 需重复 threat model            |
| CSP                            | renderer document                    | Elevation / Info disclosure | inline/eval/script/media 扩张                           | production CSP restrictive；playback 只增加 `media-src 'self' blob:`                                                         | runtime header check                   | CSP tests                                        | Electron      | dev CSP 会因 Vite 更宽                               |
| Navigation/window-open         | renderer                             | Spoofing / Elevation        | malicious link 打开 external window                     | `will-navigate`、`will-frame-navigate`、`will-redirect`、`setWindowOpenHandler deny`                                         | runtime event evidence                 | runtime smoke                                    | Electron      | future external links 需显式 allowlist               |
| shell/openExternal             | main                                 | Elevation                   | 从内容打开任意 file/url                                 | first slice 不使用                                                                                                           | source inspection                      | grep `shell/openExternal`                        | Electron      | future help links 需 allowlist                       |
| Transcript/reflections         | editor 到 disk                       | Tampering / data loss       | autosave failure 覆盖好文件                             | temp+rename；previous file preserved；retry                                                                                  | alert + hash unchanged                 | chmod save-failure test                          | Main/Frontend | app crash 前 renderer draft 可能丢失                 |
| `AGENTS.md` workspace contract | Reo 到 AI tools                      | Tampering                   | AI 删除 audio 或未经请求重写 notes                      | generated instructions 禁止删除 original audio，禁止无明确请求重写 text                                                      | Codex read-only hash diff              | Codex validation                                 | Product/QA    | external AI 可能忽略指令                             |

## 安全门禁

实现阶段不得让任何 slice 越过安全 gate，除非受影响行都具备：

- prevention；
- detection/recovery；
- 至少一个 automated 或 manual validation path。
